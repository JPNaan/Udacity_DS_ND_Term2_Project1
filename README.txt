Project Overview:
This project evaluates Seatle Airbnb data available from:  https://www.kaggle.com/airbnb/seattle/data
Files:
 - calendar.csv:  File obtained from kaggle
 - listings.csv:  File obtained from kaggle
 - reviews.csv:  File obtained from kaggle
 
 - Gather_Assess_Clean_Analyze.ipynd:  Juypter notebook that goes throught the steps to gather, assess, clean and analyze the data from the kaggle documents.  This document is ordered in accordance with the CRISP-DM process.
 
 - Model_Visualize.ipynd:  Juypter notebook that models the cleaned data, and evaluates the results
 
 - functions.py:  A function module that encapsulates the data preperation steps identified in the Gather_Assess_Clean_Analyze notebook into functions that are used in modeling and evaluating the results.  This module covers most of the data preperation step.
 
 References:
  - The following was used for feature selection reference: https://towardsdatascience.com/feature-selection-with-pandas-e3690ad8504b, by Abhini Shetye

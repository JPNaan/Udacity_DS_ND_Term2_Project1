Project Overview:
This project evaluates Seatle Airbnb data available from:  https://www.kaggle.com/airbnb/seattle/data
Files:
 - calendar.csv:  File obtained from kaggle
 - listings.csv:  File obtained from kaggle
 - reviews.csv:  File obtained from kaggle
 
 - Business-Data-Understand.ipynd:  Juypter notbook analyzing the contents of the kaggle documents
 - Analysis.ipynd:  Main project notebook that identifies and describes key findings
 - functions.py:  Function module for the functions used within the Analysis notebook
